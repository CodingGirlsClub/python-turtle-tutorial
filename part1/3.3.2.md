**Learning By Reading 难度：★ 重要性：★★★★★**

- 生成list有另外一种强大的方式“列表解析”。看[材料](http://codingpy.com/article/python-list-comprehensions-explained-visually/)。
- 比如，想要一个10个小海龟的list，我们可以用`[turtle.Turtle() for i in range(10)]`来生成。
- 但列表解析中不支持临时变量，即类似
```
for i in range(20):
    j = f(i)
    xlist.append((i+j)**j)
```
不太方便用列表解析处理。
